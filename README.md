# HireStreamAI
🤖 AI-Powered Resume Screening & Smart Job Matching Platform

## 📝 Overview
HireStreamAI is a modern AI-powered recruitment platform designed to automate resume screening and intelligent job matching using Natural Language Processing (NLP) and Machine Learning techniques.

The platform analyzes resumes, extracts candidate skills, calculates ATS compatibility scores, ranks applicants, identifies missing skills, and recommends the most suitable job opportunities in real time.

Built with a futuristic full-stack SaaS architecture, HireStreamAI combines Next.js, FastAPI, PostgreSQL, TailwindCSS, and AI-driven semantic similarity algorithms to deliver a premium recruiter and candidate experience.

## ✨ Features

📄 AI Resume Parsing  
Automatically extracts:
- Skills
- Education
- Experience
- Projects
- Candidate information

📊 ATS Score Analysis  
Calculates resume-job compatibility scores using NLP and semantic similarity matching.

🎯 Smart Job Matching  
Recommends best-fit jobs using:
- TF-IDF
- Sentence Transformers
- Cosine Similarity

🧠 Skill Gap Analysis  
Detects missing skills and provides AI-powered recommendations.

📈 Recruiter Analytics Dashboard  
Interactive dashboard with:
- Candidate ranking
- Hiring insights
- Skill analytics
- Resume comparison

🌙 Modern SaaS UI  
- Fully responsive design
- Dark & Light mode
- Glassmorphism interface
- Smooth animations
- Premium AI dashboard aesthetics
